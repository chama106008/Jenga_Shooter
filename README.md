# Jenga_Shooter

## Overview

- **Genre**: 3D First Person Shooting  
- **Engine**: Unreal Engine 5.3
- **Concept**: 初心者向け本格派FPS“体験”ゲーム  

---

## Game Design

### ゲームの仕様

外部のプラットフォーム上を移動し、ポジションを整えながら、中央のジェンガブロックを銃で撃ち抜く、シンプルなソロFPSゲーム。
特大のジェンガブロックを、飛ばして、弾いて、爆散させてハイスコアを目指す

**全力のおふざけゲーム**!!

---

## Design Philosophy

FPSゲームは、ライト層視点では「ゲーム上級者がプレイする、難しいゲーム」という評価が一般的である。
これはただの憶測に留まるものではなく、実際にFPSの中での体験というのは一般的なゲームとは一線を画すものである。
複雑な立体挙動を操作し、目まぐるしく変わる戦局の中、周囲の敵の位置を把握し、射線を見極め、遮蔽物に身を隠し、手持ちの資源を確認し、必要に応じて戦闘し、或いは逃げ、数度の戦闘の後にようやく勝利......
など、そんな難しいことを初心者が達成できるわけもなく、銃すら構えることが出来ずにどこからともなく撃たれてゲームオーバー、というのが関の山。

しかし、ゲームとはそんな過酷なものではなかったはず。
シューティングゲームと言えば、その名の通り銃を構えて、ターゲットを狙って、「Shooting」することを楽しむゲームのはず。

FPSにありがちな複雑な要素は無し、とにかく構えて、狙って、ひたすらに撃って、
そうしてジェンガをクリアするのが、このゲーム。

### Key Features
- 敵はいない、攻撃も受けない、安心して「狙って撃つ」を体験できるシンプルなゲーム性
- ダッシュ、残弾管理、「照準モード」を備えた、本格派っぽいFPS体験
- 特大スケールで、ド派手に遊ぶ「ジェンガ」

### Technical Highlights
- UE5での開発初心者のため、とにかくシンプルな設計で、最短距離で完成まで
- 後からステージ追加、ギミック追加が可能な、モジュール管理された拡張性の高い設計
